**Locked design (implement exactly; do not redesign):** Two-read detection per FINAL spec v1.0 and research artifact. Probe `read_excel_sheet(source, sheet_name, header=None)`; score each of the first `max_rows` rows by the count of `src.etl_columns.normalize_name`-normalized expected tokens present among that row's non-blank cells; select the TOPMOST row with the highest score that is `>= min_match`; raise `ValueError` naming the sheet and expected columns if none qualifies. Rewind BytesIO sources before the probe read. LE `min_match=20` (of 25 tokens); AOP `min_match=17` (of 24 tokens).
