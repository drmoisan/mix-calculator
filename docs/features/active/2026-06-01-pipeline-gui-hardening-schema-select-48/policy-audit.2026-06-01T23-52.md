## Executive Summary

The remediation makes the bundled default schemas (`default_aop`, `default_le`) first-class in
listing, loading, and matching through a single union-aware seam in `SchemaRegistry`, and wires
dropdown population at the composition root via a new small module. The change is additive: known-
file loaders and user-registry persistence are unchanged. The reviewer independently reproduced the
