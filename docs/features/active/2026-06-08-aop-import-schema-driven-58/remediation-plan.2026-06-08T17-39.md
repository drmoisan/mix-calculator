  LE/SKU_LU/orchestration/db seams. Both files end <= 500.

  the resolver-action Hypothesis property test added by #58 into a NEW module

  cohesive concern (the `resolver`/`is_tty`/`prompt` injection seam added by #58);
  the remaining tests cover the core load phases. Both files end <= 500.

Tests are moved (not duplicated); every existing test name is preserved so
coverage and AC mapping are unchanged. No assertions are weakened and no behavior
